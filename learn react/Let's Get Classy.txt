    super(props);

    };
  }	


    render() {

    }
}

React Mounts in 2 phases -
1) Render Phase
2) Commit Phase

---MOUNTING---

- Constructor(dummy)
- Render(dummy)
	- HTML (dummy)
- Component Did Mount
	- API Call
	- this.setState -> State variable is updated

---UPDATE---
- Render(API Data)
	- HTML(API Data)
- Component Did Update

--UNMOUNTING---
- Component Will Unmount
